소개

PreSonus ™ Studio One ™ 소프트웨어를 구입 해 주셔서 감사합니다.Studio One 메뉴얼 에 오신 것을 환영합니다!

Studio One 4.5은 macOS 및 Windows 용 혁신적인 음악 제작 프로그램입니다. 오디오 녹음, MIDI 시퀀싱 및 (Studio One Professional만) 오디오 마스터링을 쉽고 재미있게 할 수 있습니다.이것이 바로 음악 제작의 본연의 모습입니다. 트래킹, 믹싱,마스터링에서 전달까지, 직관적인 조작, 스피드, 효율에 배려한 크리에이티브한 제작 환경이면서, 지극히 복잡한 프로덕션에도 대응하는 견고함을 갖추고 있습니다.버전 4.5에는 폭넓은 새 기능, 기능 향상, 선진적 테크놀로지가 탑재되고 있습니다.

이 매뉴얼에는 Studio One의 조작과 자세한 기능설명이 나와 있습니다.오디오 소프트웨어에 익숙해진 분도, 처음 사용하는 분도, 이 메뉴얼로부터 읽기 시작하는 것을 추천합니다.

PreSonus에서는 지속적인 제품의 기능 향상을 위해 노력하고 있으며, 고객의 의견을 존중하고 있습니다. 계속적인 향상이라는 목적을 달성하기 위해서는, "진짜 전문가"인 중요한 유저 분들의 소리를 듣는 것이 제일이라고 믿습니다.본제품의 구입을 통한 지원에 깊은 감사를 드립니다.

Studio One버젼

Studio One 4.5에는 Artist와 Professional의 2종류의 버전이 존재하고 있습니다.모든 버전에 대해 단일 설치가 사용되고 있으며 Studio One을 액티베이트 하는 프로덕트 키에 의해 동작하는 버전이 결정됩니다. 다음으로 버전 간의 주요 차이점에 대해 설명하겠습니다.

Studio One Artist

  • 오디오트랙 및 MIDI트랙 수 무제한 동시녹음 입력 수 무제한
  • 검색 기능 및 PreSonus 샵 다이렉트 접근 기능을 탑재한 콘텐츠 브라우저
  • 컴핑, 트랜전트 검출 및 편집, 피치 보정（Melodyne 트라이얼 버전）등의 고급 편집 기능
  • SoundCloud™ 전체 통합
  • 30개의 Native Effects 5 가상 악기

Studio One Professional

  • Artist의 모든 기능과 콘텐츠를 탑재
  • 통합 마스터 링 솔루션을 제공하는 프로젝트 페이지
  • 어레인지 트랙과 스크래치 패드
  • AAF와 송 트랙 가져 오기
  • Melodyne essential 4에 의한 피치 수정 (라이센스 포함)
  • Extended FX Chains과 Multi Instrument
  • 서드파티 샘플러 포맷 지원（EXS、Giga、Kontakt、Bitwig）
  • 비디오 재생과 동기화
  • Note FX、Groove Delay、Multiband Dynamics、OpenAIR、Pipeline、IR Maker의 Native Effects플러그인
  • 서드 파티 VST / AU 플러그 인 및 ReWire 지원

VST는 Steinberg Media Technologies GmbH의 등록 상표입니다.

ASIO는 Steinberg Media Technologies GmbH의 상표 및 소프트웨어입니다.
